## UNDERTALE Engine 是什么？

这个模板能够极大地简化 UNDERTALE 粉丝游戏的制作。

需要注意的是,其中所有代码均为自主编写,没有任何原版UT中使用的代码。它基于*GameMaker*开发, 所有的提交应该基于其最新版本。

这个模板包含了战斗系统、世界系统、多语言支持等功能。部分内容是开源的,欢迎前往GitHub仓库查看!

如果您喜欢这个项目,请在[GameJolt](https://gamejolt.com/games/undertale_engine/378055)上给它加星并关注,我们不胜感激。让我们一起努力,为 UNDERTALE 粉丝游戏的发展贡献力量。

## UTEDocs 是什么？

本文档旨在整理UTE及其分支的函数文档,文档内容由社区自发维护。

我们鼓励社区成员共同参与UTE文档的整理工作,将自己在使用过程中积累的知识和经验贡献出来。文档内容将由社区志愿者不断更新和完善,以体现UTE文档的开放和共享精神。

我们欢迎更多UTE用户加入文档编写的行列。您可以修正文档中的错误,添加示例代码,补充参数解释等来丰富文档内容,帮助社区更好地理解和使用UTE。

您的每一份贡献,无论大小,都将有助于推进UTE文档建设,使其成为一个充满活力、犹如成长中的UTE般日臻完善的知识平台。

让我们团结协作,一起打造UTE文档,惠及开源社区!

## 如何进行贡献？

1. 请克隆本UTE文档项目的GitHub仓库,仓库地址可在页面右上方查看。

2. 在仓库内新建Markdown格式的文档文件。

3. 在_sidebar.md文件中添加你的新文档的相对路径,以便其出现在侧边栏目录中。

4. 提交Pull Request,由项目维护人员review后merge。
