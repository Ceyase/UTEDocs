# 准备工作

### 必须要有的软件：

- Gamemaker Studio
- Undertale Engine

### 可选的软件：

推送类软件

- Git
- Github Desktop

扩展类软件

- [GMEdit](https://yellowafterlife.github.io/GMEdit/)（第三方的Gamemaker代码编辑工具）
